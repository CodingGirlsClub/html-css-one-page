CSS 框模型: 宽度width, 内边距padding & 边框border

宽度width, 内边框padding和边框border之间的关系容易混淆.

我们一起尝试添加一个属性，并观察框的大小如何改变：（1）将宽度设置为450px，（2）添加35px的填充，（3）将边框增加到15px.

Border box example.

注意盒子的尺寸如何变宽？让我们来看看这道算式：

width + padding left + padding right + border left + border right = total size
450 + 35 + 35 + 15 + 15 = 550px
